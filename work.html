Encrypted Kubernetes Messages

Problem: Service-to-service messaging inside Kubernetes often includes sensitive payloads, but ad-hoc encryption approaches are easy to get wrong and hard to operationalize.

Solution: I built a reference implementation that explores end-to-end encrypted message exchange for Kubernetes workloads, focusing on practical integration points and reproducible demos.

Results: The project provides a concrete starting point for evaluating key management, authenticated encryption, and operational tradeoffs in secure in-cluster communication.

Tech Stack

Kubernetes, security engineering, cryptography, container tooling

ML-Pipeline Tool

Problem: Engineering teams needed an internal way to run complex LLM-assisted workflows without leaking private code to external services.

Solution: I designed a containerized pipeline that orchestrates local model interactions, iterative code operations, and controlled patching loops with clear execution boundaries.

Results: The system enabled safer automated refactoring and repeatable AI-assisted engineering workflows. It showcases end-to-end ML systems thinking, not just prompt integration.

Tech Stack

Python, Node.js, Docker, local LLM runtimes, git

MailPocket Service

Problem: Static and low-complexity web deployments often need reliable form ingestion without introducing heavy third-party dependencies.

Solution: I built a lightweight backend service in Go with clean HTTP ingestion, storage support, and container-friendly deployment for quick integration into existing infrastructure.

Results: MailPocket powers real form workflows while remaining easy to operate and extend. It demonstrates backend infrastructure pragmatism and operational simplicity.

Tech Stack

Go, SQLite, Docker, HTTP APIs
